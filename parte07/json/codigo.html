JSON

Java Script Object Notation (JSON) é um formato baseado em texto padrão para representar dados estruturados com base na sintaxe do objeto JavaScript.

É comumente utilizado para transmitir dados em aplicativos da Web (por exemplo, enviar elguns dados do servidor parao cliente, para que possam ser exibidos em uma página da Web ou vice-versa).

O JSON é um formato de dados baseado em texto seguindo a sintaxe do objeto JavaScript, que foi popularizada popularizada por Douglas Crockford.



JSON.stringify(obj) // converte para string
JSON.parse(str) // converte para objeto
